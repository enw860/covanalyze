covanalyze - Coverage analysis tool for Go coverage.out files

Usage:
  covanalyze [flags]

Flags:
  -f string        Coverage file path (required)
  -o string        Output file path (default: stdout)
  -m string        Module prefix to replace in file paths (optional, requires -mpath)
  -mpath string    Path prefix to replace module with (optional, requires -m)
  -v int           Verbose logging level (0=off, 1=info, 2=debug, 3=trace)
  --help           Show this help message

Exit Codes:
  0    Success
  1    File not found
  2    Parse error (invalid format or unsupported mode)
  3    Validation error

Examples:
  covanalyze -f coverage.out
  covanalyze -f coverage.out -o report.json
  covanalyze -f coverage.out -m github.com/user/repo -mpath ./
  covanalyze -f coverage.out -v 2
  covanalyze --help
